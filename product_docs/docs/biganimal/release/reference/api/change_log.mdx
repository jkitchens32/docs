## Notification
BigAnimal REST API follows up OpenAPI Specification (formerly Swagger Specification), all API endpoints are relative to the base URL, please see [here](https://swagger.io/docs/specification/api-host-and-base-path/) for more information.
![url-structure](./url.png)

The api changes from v1 to v2 include the following parts: the url path, the query parameters, post data schema and the response data schema. 
Based on the usage, we divide all the endpoints into 4 parts:

| Type          | Description                                          | Effect count |
| ------------- | ---------------------------------------------------- | ------------ |
| Server URL    | only change the server url from /api/v1/ to /api/v2/ | 30           |
| Endpoint path | The endpoint path is changed                         | 21           |
| Method        | GET/POST/DELETE/PUT/PATCH is changed                 | 5            |
| Added         | new endpoint added in v2                             | 9            |

## Endpoint change log
