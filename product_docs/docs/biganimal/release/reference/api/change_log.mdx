---
title: BigAnimal API v2 change log
---

BigAnimal API follows the OpenAPI Specification (formerly Swagger Specification). All API endpoints are relative to the base URL, see [API Server and Base URL](https://swagger.io/docs/specification/api-host-and-base-path/) for more information.

```
https://api.example.com/v1/users?role=admin&status=active
\________________________/\____/ \______________________/
         server URL       endpoint    query parameters
                            path
```

The summary of the API changes between v1 and v2 are:

| Type          | Description                                      | Effect count |
| ------------- | ------------------------------------------------ | ------------ |
| Server URL    | Changed server URL globally (/api/v1/ to /api/v2/)        | 30           |
| Endpoint path | Changed endpoint path                            | 21           |
| Method        | GET/POST/DELETE/PUT/PATCH                        | 5            |
| Added         | Added new endpoints                              | 9            |
 

For more information about changes to endpoint paths and methods and new added endpoints, see the following tables:
